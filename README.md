# Snake Game (Python)

A classic Snake game built using Python, featuring real-time gameplay, scoring, and multimedia effects.

## Features
- Real-time game loop
- Keyboard input handling
- Collision detection (self and boundaries)
- Score tracking and high score system
- Sound effects and background audio
- Custom graphics and assets

## Tech Stack
- Python
- Pygame (or similar game framework)

## Project Structure
- `snakegame.py` – main game logic
- `*.wav / *.mp3` – sound effects
- `*.png` – graphics/assets
- `highscore.txt` – persistent score tracking
